AN OS IS A COLLECTION OF SOFTWARES THAT ACT AS AN INTERFACE
BETWEEN A USER AND THE UNDERLYING HARDWARE.

DIFFERENT TYPES OF OS HANDLE THE HARDWARE RESOURCES IN DIFFERENT WAYS.

TYPES OF OS:

1. BATCH OS
2. MULTIPROGRAMMING OS
3. MULTI-TASKING / TIME-SHARING OS
4. REAL TIME OS
5. DISTRIBUTED OS
6. NETWORK OS
7. MOBILE OS
